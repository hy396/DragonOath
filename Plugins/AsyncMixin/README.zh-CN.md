# AsyncMixin 中文说明

## 定位

`AsyncMixin` 是 Lyra 的轻量异步加载辅助插件。

它主要用于把异步加载请求串起来，并在对象销毁、状态切换或新请求到来时安全取消旧请求，减少异步回调打到无效对象上的风险。

## DragonOath 使用建议

- UI 页面异步加载资源时可以使用。
- Common UI 推入页面前需要加载软引用 Widget 时可以使用。
- 技能图标、装备图标、角色预览资源异步加载时可以参考。

## 注意

- 它不是任务系统，也不是线程池。
- 它主要管理异步加载生命周期。
- 使用前先确认调用对象的生命周期，避免把长期业务状态藏在回调里。
