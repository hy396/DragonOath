# 00. MassAI 学习路线

## 学习目标

学完本系列后，你应能：

1. 说明 Mass Entity 与 `AActor` 怪物的职责边界。
2. 解释 Fragment、Processor、Trait、Query、Spawner 各自作用。
3. 读懂 `MassAI` 插件模块划分，知道如何启用依赖链。
4. 配置 `UMassStateTreeTrait`，理解 Mass 专用 StateTree Schema 与 Task。
5. 为 DragonOath 判断：哪些怪物场景适合 Mass，哪些必须保留 Actor + GAS。

## 前置知识

- UE 基础：Subsystem、`AActor`、组件、Tick
- 建议已了解 **StateTree** 基础（DragonOath 首期怪物会用到）
- 建议已阅读 [02_Technical_Architecture.md](../../02_Technical_Architecture.md) 中「怪物 AI / Mass + StateTree」章节
- 联机方向可先了解 [Iris 学习](../Iris_Replication_Learning/README.md)（Mass 复制与之相关但独立）
