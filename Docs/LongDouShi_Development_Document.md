# 归档：早期实验性开发文档

本文档是 DragonOath 立项前的早期实验性草稿，原内容曾用于粗略探索“横版动作 RPG / 龙斗士风格体验”的开发方向。

由于该文档存在以下问题，已不再作为正式开发依据：

- 内容属于早期探索，不符合当前 DragonOath 正式文档结构。
- 项目已经改为原创 `DragonOath / 龙契战纪` 方向。
- 部分旧内容出现编码乱码，不适合继续维护。
- 当前项目需要以 UE 5.8、GAS、联机架构优先、Common UI、Spec Kit 和 MCP/Toolset 为正式规范基础。

正式开发请以以下文档为准：

## 保留原因

保留这个文件名，是为了记录项目早期讨论来源，避免后续看到旧链接时误以为文件丢失。

## 当前结论

DragonOath 的正式方向是：

```text
原创 2.5D 横版动作 RPG
UE 5.8
GAS 核心战斗
联机架构优先
Common UI 逐步接入
工具链使用 Spec Kit + UE MCP/Toolsets
```
